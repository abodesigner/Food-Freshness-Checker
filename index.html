app main features

Date Management

You can set Best-if-used date or Expiration date for each goods.

Get Notifications

Notifications to remind you to eat products before expiry date or after opening.

Set Alarm

You can set Alarm by checking the number of days remaining displayed in the card.

Easy Search

You can find the need information with easy manner.

Shopping list

Add items to the shopping list with just a tap.

Tracking

Track opened products and those close to expiry date.

Simple Customization

Customization of categories, locations, units.

Show Summary

Show a summary with the quantity of each product in each list (also the total).

Add Images
